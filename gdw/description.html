HTML terms

HTML
HyperText Markup Language is the most basic building block of the Web.
HTML element
An HTML element is a type of HTML document component.
HTML tag
HTML tags are like keywords which defines that how web browser will format and display the content.
angle bracket
Angle brackets are commonly used to enclose a code.
HTML attribute
HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML comment
The HTML comment tag is used to add text to an HTML document without including that text in the Web page.
HTML object element
The object HTML element represents an external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.
HTML special character
HTML special characters are characters outside of the core Latin alphabet that can be represented in HTML
